# Policy Assistant 竞品调研报告

**调研时间**: 2026-02-01
**调研目标**: 企业合规、政策分析相关产品

---

## 一、竞品清单

### 1. 蚂蚁集团 - 蚁天鉴
- **定位**: 企业信用评估、风险监控
- **官网**: antfin.com
- **功能**: 企业信用评分、风险预警、司法风险
- **数据源**: 工商、司法、税务、银行等
- **目标客户**: 银行、投资机构、企业

### 2. 企查查 / 天眼查
- **定位**: 企业信息查询
- **官网**: qcc.com / tianyancha.com
- **功能**: 企业工商信息、股东、风险、司法诉讼
- **数据源**: 公开政府数据
- **目标客户**: 投资人尽调、商业情报

### 3. 启信宝
- **定位**: 企业信息与风险监控
- **官网**: qixin.com
- **功能**: 企业画像、风险监控、产业链分析
- **数据源**: 工商、司法、知识产权
- **目标客户**: 银行、金融机构

### 4. 风险雷达
- **定位**: 企业风险监控
- **官网**: riskradar.com
- **功能**: 舆情监控、风险预警、关联分析
- **数据源**: 新闻、社交媒体、政府公开数据
- **目标客户**: 风控部门、投资机构

### 5. 契约锁
- **定位**: 电子签章与合同管理
- **官网**: qiyuesuo.com
- **功能**: 电子签章、合同管理、合规审计
- **数据源**: 企业内部合同数据
- **目标客户**: 法务部门、采购部门

---

## 二、差异化分析

| 维度 | 竞品特点 | Policy Assistant 机会 |
|------|----------|----------------------|
| **数据源** | 公开数据为主 | bidata.com.cn 可能有更深度数据 |
| **AI 能力** | 基础搜索、匹配 | 深度分析、关联发现 |
| **合规检测** | 风险预警（事后） | 合规检测（事前预防） |
| **定价模式** | 高客单价（年费） | 按需付费、API 调用 |
| **目标客户** | 大企业、金融机构 | 中小企业、创业者 |

---

## 三、竞品弱点（我们的机会）

### 1. 缺乏真正的 AI 深度分析
- 企查查/天眼查：主要是数据罗列
- 蚁天鉴：评分系统，但不够透明
- **机会**: AI 驱动的深度洞察

### 2. 合规检测能力弱
- 大多是"出了风险才报警"
- **机会**: 事前预防，报告合规性检测

### 3. 中小企业市场空白
- 竞品都定位大客户
- **机会**: 低成本、易用的合规工具

### 4. 报告生成能力弱
- 大多是数据导出
- **机会**: AI 自动生成合规报告

---

## 四、我们的优势

| 优势 | 说明 |
|------|------|
| **AI 深度分析** | Agent 链式调用，能做复杂推理 |
| **规则引擎** | 可配置的合规规则库 |
| **多维度检测** | 内容、数据、风险三维度 |
| **定价计算** | 内置 SaaS 定价工具 |
| **Bot-First** | Slack 集成，协作友好 |

## 五、优化方向

### 短期（1-2周）

1. **强化 AI 分析能力**
   - 深度关联分析
   - 异常检测
   - 趋势预测

2. **增加真实数据接入**
   - 对接 bidata.com.cn API
   - 验证真实用户场景

3. **优化用户体验**
   - 简化使用流程
   - 增加模板
   - 更好的报告展示

### 中期（1个月）

1. **增加报告生成能力**
   - 自动生成合规报告
   - 支持多种格式（PDF、Word、PPT）

2. **增加预警能力**
   - 政策更新监控
   - 风险预警

3. **开放 API**
   - 支持第三方集成
   - 按调用次数收费

### 长期（3个月）

1. **行业化**
   - 针对特定行业（金融、医疗、制造）
   - 行业合规规则库

2. **生态化**
   - 与企查查、天眼查数据对接
   - 与钉钉、飞书集成

3. **智能化**
   - 持续学习用户反馈
   - 自动优化规则库
