# 产品拆解报告生成器 (Product Analyzer)

## 📖 产品说明文档

### 产品概述

**产品拆解报告生成器**是一个 AI-powered 产品分析工具，帮助用户快速了解任意产品的定位、功能、商业模式、增长策略，并提供竞品对比分析。

### 核心功能

| 功能模块 | 描述 |
|---------|------|
| **产品定位分析** | 提取产品核心价值主张和差异化定位 |
| **目标用户识别** | 分析产品的目标用户群体和细分市场 |
| **核心功能梳理** | 整理产品的主要功能和特性 |
| **商业模式拆解** | 分析收入来源、定价策略、单位经济模型 |
| **增长策略研究** | 探索产品的获客渠道和增长策略 |
| **竞品对比分析** | (新增) 对比 3-5 个核心竞品，提供差异化洞察 |

### 使用场景

1. **竞品研究**：了解竞争对手的优劣势
2. **市场调研**：快速掌握某个产品领域的市场格局
3. **产品学习**：拆解成功产品的成功要素
4. **投资分析**：评估产品的商业模式和增长潜力
5. **产品决策**：为自家产品寻找差异化机会

### 技术实现

- **输入**：产品名称、URL 或描述
- **处理**：通过 web_search 搜索产品信息、竞品信息
- **输出**：结构化的 Markdown 报告

### 竞品对比模块详解

#### 识别竞品的维度

1. **直接竞品**：目标用户相同 + 功能相似
2. **间接竞品**：解决相同问题 + 不同方案
3. **潜在颠覆者**：新兴玩家 + 大厂入局

#### 对比维度

| 维度 | 描述 |
|------|------|
| 定价策略 | 免费版、付费版、企业版定价 |
| 核心功能 | 功能完整性、独特性 |
| 目标用户 | 用户群体、公司规模 |
| 技术壁垒 | 专有技术、数据优势 |
| 市场份额 | MAU、融资、估值 |
| 增长策略 | 获客渠道、留存策略 |

#### 差异化洞察

- **本产品的差异化机会**：产品可以在哪些方面胜出？
- **竞品护城河**：哪些优势难以被复制？
- **市场空白**：未被满足的用户需求或细分市场
- **威胁预警**：新兴竞品或颠覆性趋势

### 最佳实践

1. **数据来源**：优先使用官方资料、Product Hunt、行业报告
2. **数据时效**：包含具体指标时注明来源和时间
3. **深度适中**：每个章节 2-4 句话，保持简洁但有实质内容
4. **竞品选择**：优先选择 MAU > 10M 或融资较多的竞品
5. **定价验证**：从官方渠道核实定价信息（可能变化）
6. **差异化聚焦**：突出产品 1-2 个明确的竞争优势

### 扩展建议

1. **深度分析模块**：增加技术架构分析、团队背景分析
2. **数据追踪**：保存历史报告，支持趋势对比
3. **多语言支持**：支持英文、日文等语言的产品分析
4. **模板市场**：提供不同类型产品的分析模板
5. **API 接口**：支持程序化调用和批量分析
